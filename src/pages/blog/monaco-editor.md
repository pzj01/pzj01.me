我们可能在某些网站见过在线编辑代码的功能，他们大多数使用的都是[monaco editor](https://github.com/microsoft/monaco-editor)。

## 安装

我使用的包管理器是[pnpm](https://pnpm.io/)，它的优点是安装速度快。

```bash
pnpm add monaco-editor
```
